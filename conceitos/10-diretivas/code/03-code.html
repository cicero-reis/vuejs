Diretivas são basicamente como atributos HTML que são adicionados dentro de modelos.
            Todos começam com v-, para indicar que é um atributo especial do Vue.
        
                v-html 
                Quando você usa a interpolação para imprimir uma propriedade de dados, o HTML é escapado. 
                Essa é uma ótima maneira que o Vue usa para proteger automaticamente contra ataques XSS.
                Existem casos, no entanto, em que você deseja gerar HTML e fazer com que o navegador o interprete.
